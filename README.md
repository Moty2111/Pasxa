## ✨ Что внутри?

### 🎨 CSS-иллюстрация (без картинок!)
Все элементы нарисованы с помощью CSS — градиенты, тени, псевдоэлементы, бордеры:
- **🍰 Кулич** — с глазурью, посыпкой (7 цветных конфетти), горящей свечой и анимированным пламенем
- **🥚 Три расписных яйца** — каждое с уникальным узором (ромбы, волнистые фигуры, цветочный символ ✿)
- **🐰 Кролик** — с ушками, глазками с бликами, носиком, аккуратными усами и пушистым хвостиком
- **🌿 Трава** — с текстурой, травинками и маленькими цветочками
- **🌸 Веточки вербы** — с пушистыми почками и стебельками

### ✨ Атмосферные эффекты
- **Мерцающие блёстки** — 8 частиц на фоне с плавной анимацией `floatSparkle`
- **Парящее конфетти** — 25 частиц (круги и прямоугольники) на Canvas, плавно падают сверху
- **Пульсирующая открытка** — анимация `cardGlow` меняет тень и свечение карточки
- **Мягкое сияние текста** — заголовок «ХВ» пульсирует с золотистым свечением
- **Дрожащее пламя** — свеча на куличе анимирована через `@keyframes flicker`
- **Покачивание яиц** — каждое яйцо плавно поднимается и опускается

### 🎨 Дизайн
- **Цветовая палитра:** пастельные тона — розовый, зелёный, золотистый, голубой
- **Фон:** нежный градиент от персикового к лавандовому и голубому
- **Стеклянный эффект:** `backdrop-filter: blur()` на карточке и внутренних блоках
- **Декоративная рамка:** пунктирная линия `dashed` внутри открытки
- **Типографика:** системные шрифты с акцентом на читаемость

### 📱 Адаптивность
- Медиа-запрос для экранов от 380px
- Гибкие размеры через относительные единицы
- `viewport-fit=cover` для iPhone с чёлкой
- Обработчик `orientationchange` для перестроения канваса при повороте

## 🛠 Технологии

| Технология | Назначение |
|------------|------------|
| **HTML5** | Семантическая структура |
| **CSS3** | Градиенты, тени, анимации, псевдоэлементы, CSS-иллюстрация |
| **JavaScript (ES6+)** | Canvas-анимация конфетти, обработка событий |
| **Canvas API** | Рендеринг 25 парящих частиц (круги + прямоугольники) |
| **CSS Animations** | 8 ключевых кадров: `floatSparkle`, `cardGlow`, `softPulse`, `textGlow`, `flicker`, `eggFloat` |
| **Backdrop Filter** | Эффект матового стекла на карточке |
